
# 🃏 CSS Flexbox Playing Cards Layout

A frontend web development project focused on replicating a playing card layout using pure HTML5 and modern CSS Flexbox. This project explores explicit axis alignment, flex directions, and multi-layered element positioning.

## 🚀 Features
* **Flexbox Layout Grid**: Uses a fluid, wrapping flex container (`flex-wrap: wrap`) to organize multiple cards across the screen dynamically.
* **Component-Level Alignment**: Leverages specialized alignment rules (`align-self: flex-start`, `center`, and `flex-end`) to precisely position suits across individual cards.
* **Semantic Semantic Tags**: Built cleanly with semantic tags including `<main>` and customized card block modules.

## 🛠️ How to Run
1. Name your code file exactly `index.html`.
2. Double-click the file to open it in any modern web browser (Safari, Chrome, Edge) to view the cards in action!
